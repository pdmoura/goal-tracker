# Spiritual Consistency Tracker

A lightweight **21-day consistency tracker** built just for fun because of a quick assignment for the **FCS160 course at BYU‑Idaho**. This simple web app helps you track a scripture study habit, add reflections, and celebrate completing a full 21‑day streak.

## ✅ What it does

- Tracks completion for **21 daily check-ins** (double-click a day to mark it complete)
- Stores progress and reflections in **local storage** (persists across browser sessions)
- Lets you **add notes** per day and export them as a plain text file
- Includes a celebration animation when the full 21 days are completed

## 🚀 How to use

1. Open the site
2. Double-click a day to mark it complete.
3. Single-click a day to view/edit the note for that day.
4. Use **Export Notes** to download your reflections.
5. Use **Reset** to clear all progress and start over.

## 🧱 Tech stack

- Plain **HTML + CSS + JavaScript**
- Uses **localStorage** to persist progress
- Includes **Lucide icons** and **canvas‑confetti** for simple UI enhancements
